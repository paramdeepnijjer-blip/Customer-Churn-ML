# Deploying the Churn Predictor App

## Files needed in your repo
- `app.py`
- `requirements.txt`
- `Telco-Customer-Churn.csv`

## Deploy to Streamlit Cloud (free, takes ~5 minutes)

1. Push all three files to your `Customer-Churn-ML` GitHub repo

2. Go to **share.streamlit.io** and sign in with GitHub

3. Click **"New app"**

4. Select:
   - Repository: `paramdeepnijjer-blip/Customer-Churn-ML`
   - Branch: `main`
   - Main file path: `app.py`

5. Click **Deploy** — Streamlit builds it automatically

6. You get a live link like:
   `https://paramdeepnijjer-blip-customer-churn-ml-app-xxxxx.streamlit.app`

## Add the link to your resume
In your Customer Churn project bullet, add:
`[Live Demo →](your-streamlit-link)`

## Add the link to your GitHub README
In the projects table, add the live demo link next to the repo link.

---

That's it. You now have a deployed ML product you can demo in interviews.
